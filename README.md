# SharpBladeGroundStation

某飞控地面站.

VS2015 Update3 + .Net 4.5.2

名字不提供官方翻译,自行领悟.

请勿魔改客户端.

懒得捕获异常,报异常自行重启解决.

## 配置需求

### 推荐配置

CPU:i7 7900X

GPU:GTX 1080Ti

RAM:32G

分辨率:3840*2160

### 已测试的最低配置

CPU:Z3735f

GPU:Intel HD Graphics 核心显卡

RAM:2G

分辨率:1366*768
